# Kubernetes Module

This module provisions Kubernetes clusters across supported cloud providers (AWS EKS, Azure AKS, GCP GKE). It allows configuration of cluster size, node types, networking and authentication options.

## Features
- Multi-cloud support: Works with EKS, AKS, GKE.
- Configurable cluster size: Specify number of nodes, node instance type, and scaling settings.
- Network integration: Works with networking and security modules for secure cluster networking.
- Add-ons: Optionally enable features like monitoring, logging, ingress controllers.
- Tagging/Labeling: Apply labels and tags to clusters and node pools for management and cost allocation.
