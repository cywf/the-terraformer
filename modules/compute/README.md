# Compute Module

This module provisions compute resources across supported cloud platforms. It can create and manage virtual machines/instances in AWS (EC2), Azure (VMs), and GCP (Compute Engine). It supports configuration of instance types, operating system images, network settings, and tags/labels.

## Features

- Multi-cloud support: Works across AWS, Azure, and GCP.
- Parameterized resources: Choose instance size, region, OS, and other settings via variables.
- Security integration: Integrates with networking and security modules for secure deployment.
- Auto scaling: Optionally deploy auto scaling groups or scale sets.
- Tagging/Labeling: Apply labels for resource management and cost tracking.
